Technical Milestones

Key Activities
We completed the mathematical foundation of LN-QUBO v0.1, defining the initial cost function that focused on two primary objectives: fee minimization and hop count reduction. The mathematical model was designed to encode LN routing decisions as binary variables in a QUBO matrix, where each potential path through the network could be represented as a binary choice (use path or not). This foundational work established the framework for all future model iterations.
